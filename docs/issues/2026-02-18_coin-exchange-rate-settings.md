# Summary

コイン種別間の交換レート設定と交換機能を実装する。現在、CoinType（テレビコイン、ゲームコインなど）は定義されているが、コイン同士の交換（例: テレビコイン5枚 → ゲームコイン1枚）を行う仕組みがない。ExchangeRate モデルを活用し、メインアプリ上で交換レートの設定と交換操作を可能にする。
